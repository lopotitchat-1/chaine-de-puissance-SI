#Exercice 6
##Donnée :
P = 60 kW = 60 000 W
N = 3500 tr/min
##Formules :

ω = 2π · N / 60 (rad/s)
M (couple) = P / ω (N·m)
##Calculs :

ω = 2π · 3500 / 60 = 2π · 58,333333... ≈ 366,52 rad/s

M = 60000 / 366,52 ≈ 163,64 N·m

##Résultats :

Vitesse angulaire ω ≈ 366,52 rad/s
Couple M ≈ 163,6 N·m (≈ 164 N·m)
